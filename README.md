Неофициальный перевод мода **Create: The Factory Must Grow** на русский язык.

Основной перевод взят из [официального репозитория разработчика](https://github.com/DrMango14/Create-The_Factory_Must_Grow).

[Create: The Factory Must Grow](https://www.curseforge.com/minecraft/mc-mods/create-industry)

## Установка

1. Скачайте последний релиз перевода.
2. Поместите файл в папку `resourcepacks` вашего Minecraft.
3. Активируйте ресурс-пак в настройках игры.
